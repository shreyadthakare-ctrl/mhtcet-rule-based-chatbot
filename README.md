## Project Title
MHT-CET Rule Based Chatbot

## Description
This is a rule-based chatbot developed using Python and regular expressions to guide students preparing for the MHT-CET exam.

## Features
- Exam strategy guidance
- Study timetable suggestions
- Study material recommendations
- Student query handling
- Motivational responses

## Technologies Used
- Python
- Regular Expressions
- VS Code

## How to Run
1. Install Python
2. Run `python chatbot.py`
3. Start chatting with the bot
